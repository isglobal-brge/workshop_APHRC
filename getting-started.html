Getting Started

Prerequisites & Requirements

To get the most out of this workshop, participants should have:

Technical Requirements

  • Operating System: Windows 10+, macOS 10.14+, or Ubuntu 18.04+
  • RAM: Minimum 8GB (16GB recommended)
  • Storage: At least 10GB free space
  • Internet: Stable broadband connection
  • Browser: Chrome, Firefox, or Safari (latest versions)

Knowledge Prerequisites

  • Basic understanding of R programming
  • Familiarity with RStudio interface
  • Basic statistical concepts
  • Experience with health/clinical data (helpful but not required)
  • Understanding of data privacy concepts

Pre-Workshop Setup

Complete these steps at least 3 days before the workshop:

Essential Setup Tasks

1. 📊 Install R (v4.0+) and RStudio

R Statistical Software: - Download from: CRAN - Choose the version for your operating system - Follow the installation wizard

RStudio Desktop: - Download from: Posit - Free version is sufficient - Install after R is installed

2. 📦 Install Required Packages

Run this script in RStudio to install all necessary packages:

# Install DataSHIELD packages
install.packages(c("DSI", "DSOpal", "dsBaseClient", "dsHelper"),
                 repos = c("https://cran.obiba.org", 
                          "https://cloud.r-project.org"))

# Install additional analysis packages
install.packages(c("tidyverse", "ggplot2", "tableone", 
                   "survival", "metafor"))

# Install OMOP CDM tools
install.packages("DatabaseConnector")
install.packages("SqlRender")

# Verify installation
library(dsBaseClient)
library(DSOpal)

3. 🔐 Test Server Access

  • You’ll receive credentials via email 1 week before the workshop
  • Test connection using provided script
  • Report any issues immediately to workshop-support@isglobal.org

4. 📚 Review Pre-Reading Materials

Required Reading (60 minutes total): - Introduction to DataSHIELD (30 min) - OMOP CDM Overview (20 min) - Workshop Guide (10 min)

Optional Background: - DataSHIELD: taking the analysis to the data - The Book of OHDSI

Installation Troubleshooting

Common Issues:

Problem: “Package ‘xyz’ is not available”

# Solution 1: Use alternative repository
install.packages("xyz", repos = "https://cloud.r-project.org")

# Solution 2: Install from GitHub
devtools::install_github("datashield/packagename")

Problem: “Non-zero exit status” - Update R to latest version - Install Rtools (Windows) or Xcode (Mac) - Check internet connection

Problem: “Cannot connect to Opal server” - Check VPN connection (if required) - Verify credentials - Test with ping command - Check firewall settings
